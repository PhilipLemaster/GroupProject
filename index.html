FAQ

The content isn't loading, why?

Check your internet connection and clear cookies/browser cache if encountering a 400 error.

When I search for genre, nothing appears?

Make sure you are using a genre that the API can recognize, here is a list of valid genres below. Keep in mind that most games fit into more than one of these categories..

  • Simulator
  • Quiz/Trivia
  • Fighting
  • Strategy
  • Adventure
  • Role-Playing (RPG)
  • Shooter
  • Music
  • Indie
  • Turn-based strategy (TBS)
  • Pinball
  • Platform
  • Racing
  • Sport
  • Arcade
  • Visual Novel
  • Hack and slash/Beat 'em up
  • Point-and-click

What are all these NaN tags on the games?!

The VGDB API was used as a source for the total review rating for each game, and many games have not been rated by the VGDB community. NaN represents these games.

Why do I keep seeing Command & Conquer and these other games?!

You appear to be viewing the default list of generated games, meaning that your search criteria yielded no results. Ensure that you are capitalizing the first letter of each word in your game, as the search does not pick up on lowercase titles.
